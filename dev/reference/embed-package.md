# embed: Extra Recipes for Encoding Predictors

Predictors can be converted to one or more numeric representations using
a variety of methods. Effect encodings using simple generalized linear
models
[doi:10.48550/arXiv.1611.09477](https://doi.org/10.48550/arXiv.1611.09477)
or nonlinear models
[doi:10.48550/arXiv.1604.06737](https://doi.org/10.48550/arXiv.1604.06737)
can be used. There are also functions for dimension reduction and other
approaches.
